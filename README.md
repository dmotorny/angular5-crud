# CRUD Operations -- Contacts List 
### (Angular 5.x + Firebase DB/Hosting)

<b><a target="_blank" href="https://angular5-376fa.firebaseapp.com/">LIVE DEMO</a> (Firebase hosting)</b>

## Install dependencies

<b>npm install</b>

## Start server 

<b>ng serve --open</b>

## Build production 

<b>ng build --prod</b>

## Firebase hosting deploy

<b>firebase deploy</b>
